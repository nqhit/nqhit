## 🌟 Dự án nổi bật

### [Jinstore – Hệ thống bán hàng đa nền tảng](https://github.com/nqhit/jinstore)
- **Mô tả**: Một hệ thống e-commerce với các tính năng như quản lý sản phẩm, lọc theo danh mục/giá/đánh giá, phân trang, giỏ hàng, và thanh toán real-time. Tích hợp WebSocket để cập nhật tồn kho và thông báo đơn hàng.
- **Công nghệ**: React, Redux Toolkit, Node.js, Express, MongoDB, Socket.IO, Tailwind CSS, FontAwesome.
- **Thành tựu**: Tối ưu thời gian tải trang dưới 0.5s, giảm 60% API calls nhờ React Query.

### [Admin Dashboard](https://github.com/nqhit/admin-dashboard)
- **Mô tả**: Ứng dụng quản lý người dùng với các chức năng xem, chỉnh sửa, xóa, và phân quyền. Giao diện responsive, tích hợp API với bảo mật JWT.
- **Công nghệ**: React, TypeScript, Zustand, Axios, Tailwind CSS, Vite.
- **Thành tựu**: Giảm re-render không cần thiết bằng cách sử dụng `useMemo` và `useCallback`, cải thiện hiệu suất render 25%.

### [UI Validator Library](https://github.com/nqhit/validator)
- **Mô tả**: Thư viện TypeScript cung cấp regex và Yup schema cho validation email, số điện thoại, UUID, và mật khẩu.
- **Công nghệ**: TypeScript, Yup, Jest.
- **Thành tựu**: Được sử dụng trong 3 dự án nội bộ, đảm bảo 100% dữ liệu hợp lệ trước khi gửi API.

## 🎯 Mục tiêu 2025

- **⚡ Tối ưu WebSocket**: Xây dựng reusable WebSocket hooks để đồng bộ trạng thái React với hiệu suất cao.
- **🚀 UI Toolkit**: Phát triển nội bộ UI Kit với **Tailwind + Radix + ShadCN** cho các dự án lớn.
- **🧩 Tự động hóa**: Viết script/extension để preview token, breakpoints, và lint UI tự động.
- **💎 Open-source**: Đóng góp cho các thư viện như `shadcn/ui`, `tanstack-table`, hoặc `react-query`.
- **📚 Học sâu**: Thành thạo **React Query**, **Framer Motion**, và **Next.js App Router**.
